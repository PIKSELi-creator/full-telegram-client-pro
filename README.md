# Full Telegram Client Pro 🚀

**Большой и мощный Telegram UserBot / Client** на Python с Telethon.

## Возможности
- Полноценный доступ к Telegram аккаунту
- Модульная система плагинов
- Команды: /start, /help, /info, /ping и многое другое
- Скачивание фото, видео, документов
- Автоответчик, реакции, forwarding
- Логирование и удобная конфигурация
- Легко расширять

## Быстрый старт

1. `git clone https://github.com/PIKSELi-creator/full-telegram-client-pro.git`
2. `cd full-telegram-client-pro`
3. `pip install -r requirements.txt`
4. Скопируй `.env.example` → `.env` и заполни `API_ID` и `API_HASH` (с https://my.telegram.org)
5. `python main.py`

## Структура
- `main.py` — ядро
- `plugins/` — будущие модули
- `utils/` — утилиты

Развивай проект вместе со мной!
